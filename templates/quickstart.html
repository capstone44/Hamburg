Quick Start

Introduction

Welcome! This quick start will help you get started plotting and analyzing your antenna pattern. First, we will walk you through how to plot your data, then we will explain some basic ideas behind the plot so that you understand what the plot means. Let’s get started!

Example

After the hardware system is constructed and set up for a test (see hardware guide for help with this) then you are ready to use the Antenna Measurement System website to start capturing data. To start, first click the PLOT DATA button.
SCREEN SHOT
After you click this button, you should see the Antenna Under Test (AUT) start to rotate. The AUT is sending out its signal and the receiving antenna is picking it up and storing that signal. After the AUT is done rotating you can view its signal by clicking the IMPORT DATA button.
SCREEN SHOT
After clicking the IMPORT DATA button, the signal from the rotating antenna will be plotted.
SCREEN SHOT
And that’s all there is too it. For an understanding of what the plot means, read the simple explanations bellow

Theory

First, we’ll start with the antenna pattern. Antennas send out information through pulses in electromagnetic waves. Because these waves are invisible to the eye, it is not commonly known that the electromagnetic pulses from antennas are not simply spherical shapes. In fact, their shapes are far more complicated and beautiful than a sphere. Several examples of antenna patters can be seen in the bellow figures.
