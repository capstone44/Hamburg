Quick Start

Introduction

Welcome! This quick start will help you get started plotting and analyzing your antenna pattern. First, it will walk you through how to plot your data, then it will explain some basic ideas behind the plot so that you understand what the plot means. Let’s get started!

Example

After the hardware system is constructed and set up for a test (see hardware guide for help with this) you are ready to use the Antenna Measurement System website to start a capturing data. To start, click the PLOT DATA button as shown in figure 1.
FIGURE 1
After you click this button, you should see the Antenna Under Test (AUT) start to rotate. The AUT is sending out its signal and the receiving antenna is measuring the power of the signal and storing the values. After the AUT is done rotating you can view its signal by clicking the IMPORT DATA button shown in figure 2.
FIGURE 2
After clicking the IMPORT DATA button, the signal from the rotating antenna will be plotted. This is shown in figure 3.
FIGURE 3
And that’s all there is too it. To understand how to analyze the plot, read the simple explanation bellow.

Theory

What is an antenna pattern? Antennas send out information through electromagnetic waves. Because these waves are invisible to the eye, a misconception is that the electromagnetic radiation emmits from antennas in a spherical shape. In fact, radiation shapes are far more complicated and beautiful than a sphere. These radiation shapes are called antenna patterns. Several examples of antenna patters can be seen in figures 4-6.
